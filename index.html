Em Javascript, se você tentar somar qualquer numero com uma string, o número será transformado em String também. Por isso é necessário ficar bem atento quanto aos tipos de dados do JavaScript.(bugado)

Vamos tentar fazer uma soma maluca de um numero com a string JavaScript, insira o Numero:

+ JavaScript =

Os tipos de dados em JavaScript são bastantes dinâmicos, você pode por exemplo simplesmente declarar uma variável sem um tipo de dado(ele ficará undefined) e poderá alterar depois, não é recomendado fazer isso de acordo com as boas práticas de programação.

var = x;

var x = 5;

var x = "Joao"

Uma variavel que será string recebe seu dado com aspas simples ou aspas duplas

Exemplos:

var x = "João"

var x = 'João'

Isso não vai influenciar em nada, mas pelas boas práticas de programação, sempre que for colocar um dado em uma string o recomendável é utilizar aspas duplas (" ").

Para referenciar um dado que em sua string contenha aspas, exemplo: Anderson "Spider" Silva, devemos nesse caso, sobrepor as aspas, ficaria mais ou menos assim: var x = 'Anderson "Spider" Silva'

Anderson Spider Silva

Em JavaScript também existem os operadores Boleanos:

Exemplo:

var x = 5;

var y = 5;

var z = 6;

(x == y) Return true

(x == z) Return false
