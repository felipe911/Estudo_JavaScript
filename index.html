Strings

Vamos tentar fazer uma soma maluca de um numero com a string JavaScript, insira qualquer número:

+ JavaScript =

Uma variável que será string recebe seu dado com aspas simples ou aspas duplas

Exemplos:

var x = "João"

var x = 'João'

* Isso não vai influenciar em nada, mas pelas boas práticas de programação, sempre que for colocar um dado em uma string o recomendável é utilizar aspas duplas (" ").

Para referenciar um dado que em sua string contenha aspas, exemplo: Anderson "Spider" Silva, devemos nesse caso, sobrepor as aspas, ficaria mais ou menos assim: var x = 'Anderson "Spider" Silva'

Diferenças entre o uso do onClick e addEventListener

A grande diferença entre os dois é que com o onClick, só se pode passar 1 função por vez, já com o addEventListener podemos passar mais de uma função, atribui um manipulador de eventos ao elemento especificado. Exemplo de Código:

document.getElementById("meuBotao").addEventListener("click", funcao1);

document.getElementById("meuBotao").addEventListener("click", funcao2);
